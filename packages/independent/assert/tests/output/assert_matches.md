## works

```js
assert({
  actual: {
    a: "expired 3 seconds ago",
    b: true,
  },
  expect: {
    a: assert.matches(/expired \d seconds ago/),
    b: false,
  },
});
```

```console
AssertionError: actual and expect are different

actual: {
  a: "expired <X> second ago",
  b: true,
}
expect: {
  a: assert.matches(/expired \d seconds ago/),
  b: false,
}
```
